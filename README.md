# Harshit Goel Portfolio

A modern and responsive portfolio website built using **React.js** to showcase my projects, skills, and experience.

Live Demo: https://harshitgoel-portfolio.vercel.app/

---

## Overview

This portfolio is designed to highlight my development journey, projects, and technical skills.  
It features a clean UI, smooth animations, and a well-structured layout.

---

## Features

- Modern UI with clean design  
- Fully responsive (mobile-first approach)  
- Smooth animations and transitions  
- Dynamic project showcase  
- Contact section integration  

---

## Tech Stack

- React.js  
- JavaScript  
- Tailwind CSS  
- Vercel (Deployment)
